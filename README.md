## 自定义事件的监听和绑定
   * 绑定事件监听  事件名/回调
   * 事件的触发(emit)/发布(dispatch)   事件名/数据
## DOM事件
   * 绑定事件监听  事件名/回调
   * 用户操作触发事件   事件名/数据（event就是包含所有数据的一个对象）
## vue中的自定义事件的绑定和监听
   * v-on:xx='fn'  父组件-->子组件 绑定在子组件上
   * this.$emit('xx', data)   子组件进行事件的触发/发布
